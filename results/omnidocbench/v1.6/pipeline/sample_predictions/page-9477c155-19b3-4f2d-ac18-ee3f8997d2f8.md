## G.1.3. BOUNDS ON THE SECOND MOMENT ALONG THE FINITE-PARTICLE SYSTEMS

In this section, we show that along the finite-particle dynamics (16) and algorithm (20), the distributions remain in $\mathcal { P } ( \mathbb { R } ^ { 2 d N } )$

Under Assumption1, recall from Theorem35 (in Section H) that there exists a unique equilibrium point $z ^ { * } = ( x ^ { * } , y ^ { * } ) \in \mathbb R ^ { 2 d }$ which satisfies $\nabla V ( z ^ { * } ) = 0$ Define $\mathbf { z } ^ { * } = ( x ^ { * } , \ldots , x ^ { * } , y ^ { * } , \ldots , y ^ { * } ) \in$ $\mathbb { R } ^ { 2 d N }$ . Then by construction, $b ^ { \mathbf { Z } } ( { \mathbf { z } } ^ { * } ) = 0$ , where $b ^ { \mathbf { Z } }$ is the vector field defined in (17).

Lemma 31 Assume Assumption 1. Suppose $\mathbf { Z } _ { t } \sim \rho _ { t } ^ { \mathbf { Z } }$ evolves following the finite-particle dynamics (16) in R2dN from $\mathbf { Z } _ { 0 } \sim \rho _ { 0 } ^ { \mathbf { Z } } \in \mathcal { P } ( \mathbb { R } ^ { 2 d N } )$ . Then $\rho _ { t } ^ { \mathbf { Z } } \in \mathcal { P } ( \mathbb { R } ^ { 2 d N } )$ for all $t \geq 0$

Proof We note that $\rho _ { t } ^ { \mathbf { Z } }$ is absolutely continuous with respect to the Lebesgue measure on $\mathbb { R } ^ { 2 d N }$ by virtue of the Brownian motion component in the dynamics (16). We now show that the second moment of $\rho _ { t } ^ { \mathbf { Z } }$ remains finite for all $t \geq 0$ Since $\mathbf { Z } _ { t }$ evolves following the dynamics (16), its density $\rho _ { t } ^ { \mathbf { Z } }$ evolves following the Fokker-Planck equation:

$$
\frac { \partial \rho _ { t } ^ { \mathbf { Z } } } { \partial t } = - \nabla \cdot \left( \rho _ { t } ^ { \mathbf { Z } } b ^ { \mathbf { Z } } \right) + \tau \Delta \rho _ { t } ^ { \mathbf { Z } } .
$$

From this, we can compute, using integration by parts:

$$
\begin{array} { r l } { \frac { d } { d t } \mathbb { E } _ { p _ { \xi } ^ { \prime } } \Big [ \| Z _ { \xi } - z ^ { * } \| ^ { 2 } \Big ] = \int _ { \mathbb { R } ^ { 2 N } } \frac { \partial \rho _ { \xi } ^ { \prime } \{ z _ { \xi } \} } { \partial t } \| z - z ^ { * } \| ^ { 2 } d z } & { } \\ & { = \int _ { \mathbb { R } ^ { 2 N } } \big ( - \nabla \cdot ( \rho _ { \xi } ^ { 2 } \bar { p } ^ { 2 } ) \{ z \} + \gamma \Delta \rho _ { \xi } ^ { \prime } ( z ) \big ) \| z - z ^ { * } \big \| ^ { 2 } d z } \\ & { = \int _ { \mathbb { R } ^ { 2 N } } \rho _ { \xi } ^ { \prime } ( z ) \langle \bar { p } ^ { 2 } ( z ) , \nabla \cdot ( | [ z - z ^ { * } ] | ^ { 2 } \rangle ) \ d z + \int _ { \mathbb { R } ^ { 2 N } } \rho _ { \xi } ^ { \prime } ( z ) \Delta \left( | [ z - z ^ { * } ] | ^ { 2 } \right) d z } \\ & { = 2 \int _ { \mathbb { R } ^ { 2 N } } \rho _ { \xi } ^ { \prime } ( z ) \langle \bar { p } ^ { 2 } ( z ) , z - z ^ { * } \rangle \ d z + \int _ { \mathbb { R } ^ { 2 N } } \rho _ { \xi } ^ { \prime } ( z ) \langle d ( X ) \rangle d z } \\ & { = 2 \int _ { \mathbb { R } ^ { 2 N } } \rho _ { \xi } ^ { \prime } ( z ) \langle \bar { p } ^ { 2 } ( z ) - b ^ { 2 } ( z ) , z - z ^ { * } \rangle \ d z + 4 \pi d N } \\ & { \leq - 2 \alpha \int _ { \mathbb { R } ^ { 2 N } } \rho _ { \xi } ^ { \prime } ( z ) \| z - z ^ { * } \| ^ { 2 } d z + 4 \pi d N } \\ & { = - 2 \alpha \mathbb { E } _ { p _ { \xi } ^ { \prime } } \left[ \| z _ { \xi } - z ^ { * } \| ^ { 2 } \right] + 4 \pi d N } \end{array}

where the inequality above uses the property that $- b ^ { \mathbf { Z } }$ is α-strongly monotone, from Lemma 12. We can write the differential inequality above equivalently as:

$$
\frac { d } { d t } \left( e ^ { 2 \alpha t } \mathbb { E } _ { \rho _ { t } ^ { \mathbf { Z } } } \left[ \left. \mathbf { Z } _ { t } - \mathbf { z } ^ { * } \right. ^ { 2 } \right] \right) \leq e ^ { 2 \alpha t } 4 \tau d N .
$$

Integrating from 0 to t and rearranging the result gives:

$$
\begin{array} { r l } & { \mathbb { E } _ { \rho _ { t } ^ { \mathbf { Z } } } \left[ \left\| \mathbf { Z } _ { t } - \mathbf { z } ^ { * } \right\| ^ { 2 } \right] \leq e ^ { - 2 \alpha t } \mathbb { E } _ { \rho _ { 0 } ^ { \mathbf { Z } } } \left[ \left\| \mathbf { Z } _ { 0 } - \mathbf { z } ^ { * } \right\| ^ { 2 } \right] + \frac { \left( 1 - e ^ { - 2 \alpha t } \right) } { \alpha } 2 \tau d N } \\ & { \qquad \leq \mathbb { E } _ { \rho _ { 0 } ^ { \mathbf { Z } } } \left[ \left\| \mathbf { Z } _ { 0 } - \mathbf { z } ^ { * } \right\| ^ { 2 } \right] + \frac { 2 \tau d N } { \alpha } . } \end{array}
