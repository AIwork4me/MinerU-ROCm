<1>区间 \( \left[-\frac{5\pi}{6},\frac{2\pi}{3}\right] \) 上单调递增

\[
\begin{array}{l} x \in [ - \frac {5 \pi}{6}, \frac {2 \pi}{3} ] \Rightarrow \omega x + \frac {\pi}{6} \in [ \frac {\pi}{6}, \frac {5 \pi \omega}{6} + \frac {\pi}{6} ] \\ \uparrow \Rightarrow \left\{ \begin{array}{l} - \frac {\pi}{2} \leq - \frac {5 \pi}{6} \omega + \frac {\pi}{6} <   \frac {2 \pi}{3} \omega + \frac {\pi}{6} \leq \frac {\pi}{2} \\ \left| \frac {2 \pi}{3} - \left(- \frac {5 \pi}{6}\right) \right| \leq \frac {T}{2} \end{array} \right\} \Rightarrow \left\{ \begin{array}{l} \omega \leq \frac {4}{5} \\ \omega \leq \frac {2}{3} \Rightarrow 0 <   \omega \leq \frac {1}{2} \\ \omega \leq \frac {1}{2} \\ \omega > 0 \end{array} \right. \end{array}
\]

<2>存在唯一 \(x_0 \in [0, \frac{5\pi}{6}]\)，使得 \(f(x_0) = 1\)，即 \(\omega x + \frac{\pi}{6} \in [\frac{\pi}{6}, \frac{5\pi\omega}{6} + \frac{\pi}{6}]\) 内有一条对称轴 \(\therefore \frac{\pi}{2} \leq \frac{5\pi\omega}{6} + \frac{\pi}{6} < \frac{5\pi}{2}\)，得 \(\frac{2}{5} \leq \omega < 3\)，综上，有 \(\frac{2}{5} \leq \omega \leq \frac{1}{2}\).
